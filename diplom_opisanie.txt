Веб приложение: для изучения иностранных слов
Цель проекта: Разработать веб-приложения для изучения иностранных слов, с возможностью добавления в словарь и последующего их повторения.

Функционал:
-создание личного кабинета через e-mail
-дать возможность пользователю добавлять новые слова в свой словарь через поиск и сохранять их в базу данных
-дать возможность пользователю добавлять новые папки, изменять их название
-реализовать функцию напоминания пользователю о повторении слов через эл почту
    Дополнение:
    1. Добавить учетную запись преподавателя(с возможностью добавления слов в словарь ученика).
    2. Добавление картинок к переводу слов.
    3. Добавление примеров к переводу слов.
    4. аудио воспроизвидение слов
    5. Настроить переодичность повторения для каждого слова или папки.

Ресурсы:
    - API Яндекс.Переводчика
    - Google Cloud Platform
    - The Cambridge Dictionary API
    - или библиотека Googletrans
    - ABBYY Lingvo 

Задачи:
    - написать функцию которая на базе requests будет получать данные по API
    - создание домашней страницы на Flask, с описание сайта и приглашением для регистрации пользователя либо ввода своих данных
    - чат, репозиторй, трелло
